# LooperDo - Adaptive Exam Prep Platform Website

## Project Overview
Marketing + product website for LooperDo, an adaptive exam prep engine built with LangGraph/FastAPI on AWS Lambda. This NextJS frontend serves as the public-facing website with mock API integration, ready for real backend connection.

## Brand & Positioning
- **Brand Name**: LooperDo
- **Tagline**: "Know exactly when you're ready to pass"
- **Positioning**: Lead with OUTCOMES, not AI. Focus on certainty, readiness, confidence.
- **Core Concept - The Flywheel**: Generate Test → Take Test → AI Diagnosis → Study Workbook → Repeat → Ready to Pass
- **Key Differentiator**: Exam Readiness Score - mathematically grounded, per sub-topic proficiency tracking

## Visual Style
- **Color Palette**: Professional corporate blues (#1e3a5f primary dark, #2563eb primary blue, #3b82f6 accent blue), grays, white backgrounds
- **Accent**: Teal/green for success states (#10b981), orange for warnings (#f59e0b)
- **Typography**: Clean, modern sans-serif (system fonts via Tailwind)
- **Design Language**: Clean, corporate, trustworthy. Subtle shadows, rounded corners ≤5px, smooth animations
- **Animations**: Framer Motion for scroll reveals, readiness gauge animation, flywheel diagram

## Page Structure
1. **Landing** (`/`) - Hero, animated readiness gauge, flywheel visual, benefits, testimonials, CTAs
2. **Certifications** (`/certifications`) - Grid of supported certs with detail pages
3. **Dashboard** (`/dashboard`) - Protected. Readiness score, test history, sub-topic breakdown
4. **Pricing** (`/pricing`) - 3 tiers: Free, Pro ($29/mo), All-Access ($49/mo)
5. **Auth** (`/login`, `/signup`) - Email/password auth
6. **Profile** (`/profile`) - User settings
7. **About** (`/about`) - Mission, how it works
8. **FAQ** (`/faq`) - Common questions
9. **Contact** (`/contact`) - Contact form

## Supported Certifications
AWS SAA, Lean Six Sigma Black Belt (IASSC), Azure AZ-104, Power BI PL-300, GCP, PMP

## Technical Patterns
- NextAuth v4 with Prisma adapter for auth
- PostgreSQL via Prisma ORM
- Mock API endpoints simulating real FastAPI backend
- Server/client component separation enforced
- Framer Motion for animations (client-only)
- Recharts for data visualization

## Pricing Tiers
- Free: 1 certification, limited tests
- Pro ($29/mo): 1 certification, unlimited tests + workbooks
- All-Access ($49/mo): All certifications, unlimited everything
